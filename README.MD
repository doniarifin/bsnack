## Stack

1. Go 1.23.3
2. PostgreSQL version 16.10
3. Redis 7.0.15

## Quick start

1. copy .env_copy to .env

```
cp .env_copy .env
```

2. install depedencies & run app

```
go mod tidy
```

```
go run main.go
```
